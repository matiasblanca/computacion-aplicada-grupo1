## Contenido del repositorio

Este repositorio contiene los directorios generados desde la máquina virtual Debian utilizada como servidor ('TPServer'):
- root.tar.gz: directorio /root
- etc.tar.gz: directorio /etc
- opt.tar.gz: directorio /op
- www_dir.tar.gz : directorio /www_dir
- backup_dir.tar.gz: directorio /backup_dir
- var_part_: partes del archivo var.tar.gz generados a partir del directorio /var, dividido en archivos mas pequeños usando el comando split

Los archivos fueron generados con el comando tar en la máquina virtual y subidos a este repositorio mediante git sobre SSH
